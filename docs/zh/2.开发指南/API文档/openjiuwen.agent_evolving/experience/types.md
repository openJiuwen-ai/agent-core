# openjiuwen.agent_evolving.experience.types

在线经验生命周期编排所用的显式中间类型。

## class EvolutionContext

生成经验所需的在线/离线 canonical 输入。

## class PendingChange

等待用户审批的暂存演进记录快照。

## class ExperienceProposal

写入或审批前生成的经验提案。

## class ExperienceApprovalRequest

暂存经验提案的审批视图。

## class ExperienceApplyResult

应用暂存经验变更的结果。
