# indexing

`openjiuwen.core.retrieval.indexing` 提供了文档索引构建的抽象接口和实现，包括索引管理器和文档处理器。

**详细 API 文档**：[indexer/README.md](./indexer/README.md)、[processor/README.md](./processor/README.md)

**Modules**：

| MODULE | DESCRIPTION | 详细 API |
|--------|-------------|----------|
| [indexer](./indexer/README.md) | 索引管理器模块，提供索引的创建、更新和删除功能。 | [indexer/README.md](./indexer/README.md) |
| [processor](./processor/README.md) | 文档处理器模块，提供文档解析、分块、提取和分割功能。 | [processor/README.md](./processor/README.md) |
