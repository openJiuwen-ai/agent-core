# chunker

`openjiuwen.core.retrieval.indexing.processor.chunker` 提供了文本分块器的抽象接口和实现。

**Classes**：

| CLASS | DESCRIPTION | 详细 API |
|-------|-------------|----------|
| **Chunker** | 文本分块器抽象基类。 | [base.md](./base.md) |
| **CharChunker** | 基于字符的分块器。 | [char_chunker.md](./char_chunker.md) |
| **TokenizerChunker** | 基于 tokenizer 的分块器。 | [tokenizer_chunker.md](./tokenizer_chunker.md) |
| **TextChunker** | 文本分块器（支持字符/token）。 | [chunking.md](./chunking.md) |
