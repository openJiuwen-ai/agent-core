# openjiuwen.agent_evolving.dataset

`openjiuwen.agent_evolving.dataset` provides data types and capabilities for loading, shuffling, and splitting samples for self-evolving training.

---

## class openjiuwen.agent_evolving.dataset.case.Case

Single training/evaluation sample (Pydantic model).

* **inputs**(Dict[str, Any]): Input data (e.g., query or conversation content), at least one key-value pair.
* **label**(Dict[str, Any]): Expected answer or output, at least one key-value pair.
* **tools**(List[ToolInfo], optional): List of tools available for this sample. Default: `None`.
* **case_id**(str, optional): Sample unique identifier; auto-generated if not provided. Default: Generated by `uuid.uuid4().hex`.

---

## class openjiuwen.agent_evolving.dataset.case.EvaluatedCase

Evaluation result with model output and score.

* **case**(Case): Original Case.
* **answer**(Dict[str, Any], optional): Model output/prediction. Default: `None`.
* **score**(float): Comprehensive score, value range [0, 1]; clipped to this range during validation. Default: `0.0`.
* **reason**(str): Scoring reason or error analysis. Default: `""`.
* **per_metric**(Dict[str, float], optional): Per-metric scores when using MetricEvaluator. Default: `None`.

The underlying `case` fields can be accessed via proxy properties `inputs`, `label`, `tools`, `case_id`.

---

## class openjiuwen.agent_evolving.dataset.case_loader.CaseLoader

Wrapper for Case list, supporting iteration and proportional splitting.

```text
class openjiuwen.agent_evolving.dataset.case_loader.CaseLoader(cases: List[Case])
```

**Parameters**:

* **cases**(List[Case]): Case list to wrap.

### __len__() -> int

Returns the number of samples.

### __iter__() -> Iterator[Case]

Iterates through Cases in order.

### get_cases() -> List[Case]

Returns a copy of the internal Case list.

### split(ratio: float, seed: int = 0) -> Tuple[CaseLoader, CaseLoader]

Splits samples into two parts by ratio (shuffles first by seed, then splits).

**Parameters**:

* **ratio**(float): Split ratio, value range [0.0, 1.0].
* **seed**(int, optional): Random seed for reproducibility. Default: `0`.

**Returns**:

**(CaseLoader, CaseLoader)**, first and second CaseLoaders.

**Exceptions**:

* **ValueError**: Raised when `ratio` is not in [0.0, 1.0].

---

## func openjiuwen.agent_evolving.dataset.case_loader.shuffle_cases(cases, seed=0)

Shuffles a Case list with given seed, does not modify the original list.

**Parameters**:

* **cases**(List[Case]): List to shuffle.
* **seed**(int, optional): Random seed. Default: `0`.

**Returns**:

**List[Case]**, new shuffled list.

---

## func openjiuwen.agent_evolving.dataset.case_loader.split_cases(cases, ratio)

Splits a Case list into two parts by ratio.

**Parameters**:

* **cases**(List[Case]): List to split.
* **ratio**(float): Ratio, value range [0.0, 1.0].

**Returns**:

**Tuple[List[Case], List[Case]**, first and second parts.

**Exceptions**:

* **ValueError**: Raised when `ratio` is not in [0.0, 1.0].
