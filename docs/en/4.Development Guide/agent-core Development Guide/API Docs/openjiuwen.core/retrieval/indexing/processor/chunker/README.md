# chunker

`openjiuwen.core.retrieval.indexing.processor.chunker` provides abstract interfaces and implementations for text chunkers.

**Classes**:

| CLASS | DESCRIPTION | Detailed API |
|-------|-------------|---------------|
| **Chunker** | Text chunker abstract base class. | [base.md](./base.md) |
| **CharChunker** | Character-based chunker. | [char_chunker.md](./char_chunker.md) |
| **TokenizerChunker** | Tokenizer-based chunker. | [tokenizer_chunker.md](./tokenizer_chunker.md) |
| **TextChunker** | Text chunker (supports character/token). | [chunking.md](./chunking.md) |
