# openjiuwen.agent_teams

`openjiuwen.agent_teams` provides the API documentation for the AgentTeams multi-agent framework.

| Module | Description |
|---|---|
| [`agent_teams`](./openjiuwen.agent_teams/agent_teams.md) | Public interfaces for AgentTeams. |
| [`schema`](./openjiuwen.agent_teams/schema/schema.md) | AgentTeams configuration schema (`DeepAgentSpec`, supporting specs, and more). |
| [`deep_agent_spec`](./openjiuwen.agent_teams/schema/deep_agent_spec.md) | Detailed configuration schema for `DeepAgent`. |
