# indexing

`openjiuwen.core.retrieval.indexing` provides abstract interfaces and implementations for document indexing, including index managers and document processors.

**Detailed API Documentation**: [indexer/README.md](./indexer/README.md), [processor/README.md](./processor/README.md)

**Modules**:

| MODULE | DESCRIPTION | Detailed API |
|--------|-------------|---------------|
| [indexer](./indexer/README.md) | Index manager module, providing index creation, update, and deletion functionality. | [indexer/README.md](./indexer/README.md) |
| [processor](./processor/README.md) | Document processor module, providing document parsing, chunking, extraction, and splitting functionality. | [processor/README.md](./processor/README.md) |
