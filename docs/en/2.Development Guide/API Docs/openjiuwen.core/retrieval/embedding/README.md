| **APIEmbedding** | API embedding model implementation. | [api_embedding.md](./api_embedding.md) |
| **OllamaEmbedding** | Ollama embedding model implementation. | [ollama_embedding.md](./ollama_embedding.md) |
| **OpenAIEmbedding** | OpenAI embedding model implementation. | [openai_embedding.md](./openai_embedding.md) |
| **VLLMEmbedding** | vLLM embedding model implementation. | [vllm_embedding.md](./vllm_embedding.md) |
