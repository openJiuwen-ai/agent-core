# openjiuwen.agent_evolving.checkpointing.skill_package

Skill directory pack/unpack and `skill_id` frontmatter management. These functions are exported by the sharing module for obtaining skill_id and package bytes during first skill package upload.

---

## func openjiuwen.agent_evolving.checkpointing.skill_package.read_skill_id_from_content(content) -> str

Parses and reads the `skill_id` field from the top-level frontmatter in SKILL.md file content. Returns an empty string when the field is not present in the frontmatter.

**Parameters**:

* **content**(str): Full text content of the SKILL.md file.

**Returns**:

**str**, the parsed skill_id; returns `""` when no `skill_id` field exists in the frontmatter.

---

## func openjiuwen.agent_evolving.checkpointing.skill_package.ensure_skill_id_in_content(content) -> Tuple[str, str]

Ensures the SKILL.md file content's frontmatter contains a `skill_id` field. If `skill_id` is already present in the frontmatter, returns the content and existing ID unchanged; otherwise auto-generates a globally unique ID with `sk_` prefix and inserts it into the frontmatter — if a frontmatter block exists, a `skill_id:` line is appended inside the block; if no frontmatter block exists, a new one is created at the beginning of the file.

**Parameters**:

* **content**(str): Full text content of the SKILL.md file.

**Returns**:

**Tuple[str, str]**, `(updated_content, skill_id)` tuple. Returns original content and existing ID when skill_id is already present; returns updated content with the inserted skill_id and the auto-generated ID otherwise.

**Example**:

```python
>>> from openjiuwen.agent_evolving.sharing import ensure_skill_id_in_content, read_skill_id_from_content
>>>
>>> content = "---\nname: bash_tool\n---\n\n# Bash Tool\n"
>>> updated, skill_id = ensure_skill_id_in_content(content)
>>> print(skill_id)
>>> print(read_skill_id_from_content(updated))
sk_xxxxxxxxxxxx
sk_xxxxxxxxxxxx
```

## func openjiuwen.agent_evolving.checkpointing.skill_package.pack_skill_directory(skill_dir, *, skill_md_relpath=None, skill_md_content=None) -> bytes

Tar-gzips a skill directory into bytes. Automatically excludes evolution-local artifacts: `evolution/`, `archive/`, `__pycache__/`, `.git/` directories, `evolutions.json` file, and all hidden files (starting with `.`). When both `skill_md_relpath` and `skill_md_content` are provided, the tarball uses the supplied alternative content for the file at the matching path instead of the on-disk original — hub sharing leverages this to exclude locally projected evolution index blocks, ensuring the uploaded skill package does not contain evolution indexes that are only valid locally.

**Parameters**:

* **skill_dir**(Path): Absolute or relative path of the skill directory, used as the packaging root.
* **skill_md_relpath**(str, optional): Relative path of SKILL.md within the skill directory (e.g., `"SKILL.md"`), used together with `skill_md_content`; the file at this path in the tarball will be replaced by the alternative content. Default: `None`.
* **skill_md_content**(str, optional): Alternative SKILL.md text content; only effective when `skill_md_relpath` is also provided; when `None`, the on-disk original file is used. Default: `None`.

**Returns**:

**bytes**, complete tar.gz package bytes, ready for hub upload or local extraction.

**Example**:

```python
>>> from pathlib import Path
>>> from openjiuwen.agent_evolving.sharing import pack_skill_directory, unpack_skill_package
>>>
>>> skill_dir = Path("/path/to/skills/bash_tool")
>>> package_bytes = pack_skill_directory(skill_dir)
>>> print(len(package_bytes))
1024
```

## func openjiuwen.agent_evolving.checkpointing.skill_package.unpack_skill_package(package_bytes, dest_dir) -> None

Extracts a skill package tarball into the destination directory. The directory is created automatically if it doesn't exist (including parent directories). Extraction uses the `data_filter` security filter on Python 3.12+ to prevent path traversal and other security risks; falls back to traditional extraction on Python versions that don't support `data_filter`.

**Parameters**:

* **package_bytes**(bytes): tar.gz package bytes generated by `pack_skill_directory`.
* **dest_dir**(Path): Destination extraction directory path; created automatically if it doesn't exist.

**Example**:

```python
>>> from pathlib import Path
>>> from openjiuwen.agent_evolving.sharing import pack_skill_directory, unpack_skill_package
>>>
>>> skill_dir = Path("/path/to/skills/bash_tool")
>>> package_bytes = pack_skill_directory(skill_dir)
>>> unpack_skill_package(package_bytes, Path("/tmp/installed_skills/bash_tool"))
```
