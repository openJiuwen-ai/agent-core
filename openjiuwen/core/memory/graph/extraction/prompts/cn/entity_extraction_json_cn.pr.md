# 你的任务
请基于数据源描述和JSON内容，从JSON中提取相关的实体。对每个提取的实体，根据提供的实体类型及其描述，确定其所属实体类型，并输出对应的entity_type_id。

## 注意事项
1. **主实体提取**
   - 始终优先提取JSON所表示的核心实体（例如字段 `"name"`、`"user"` 等关键标识）。
   - 如果JSON表示一个对象集合，应确保每个对象的主实体都能被提取。

2. **事件提取**
   - 尽量将与人类用户相关的事件提取为实体。
   - 事件实体的命名需要足够独特，这样便于区分。

3. **排除规则**
   - 不要提取日期、时间或其他与时间相关的信息（这些由其他模块单独处理）。

4. **消歧与命名**
   - 对代词或模糊指代（如 *“他/她/它/老李/小张/这个/那些”*）进行消歧，并替换为其明确的、被引用的实体名称。
   - 命名必须清晰、唯一且无歧义；如能使用全名或完整标识，请优先使用。
   - 避免使用不完整或上下文依赖性过强的称呼。{{extra_message}}
