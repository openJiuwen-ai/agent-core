运行一个 swarmflow 编排脚本（多 agent 工作流）。

## 何时调用
- 用户要求用 swarmflow / workflow 跑一个脚本，或给出了脚本路径；
- 用户描述的任务需要多个 agent 并行 / 流水线编排，且已指定脚本。

## 行为契约
- 该工具**立即返回**——工作流在后台异步执行，**不要轮询**等待结果。
- 各阶段（phase）进展会**自动**作为通知流式进入你的上下文。
- 工作流**完成或失败时，最终结果会自动回灌**给你，无需主动查询。
- 你处于**旁观角色**：脚本自主编排 worker 完成工作，你只需在收到阶段进展通知时，用简洁自然语言向用户汇报当前进展。
- **不要**自己 spawn 成员、创建任务或尝试代替脚本编排——编排完全由脚本负责。
